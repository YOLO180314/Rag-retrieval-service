# 平台规则 FAQ

**平台的营业时间是什么？**
平台是 24 小时线上运营的，您可以随时浏览商品、下单购物。但客服人工服务时间为每天 9:00-22:00，在此之外的咨询将由 AI 智能客服处理并留言转交人工客服次日回复。商家的发货时间通常为工作日（周一至周五）的 9:00-18:00。

**订单支付超时是多长时间？**
订单提交后需在 30 分钟内完成支付。超时未支付的订单将被系统自动取消，商品库存释放回系统。如果您在支付过程中遇到问题，建议在倒计时结束前取消订单重新提交，避免超时。

**平台如何保障个人信息安全？**
平台非常重视您的个人信息安全：
1. 数据传输全程使用 HTTPS 加密
2. 密码使用不可逆加密算法存储，平台也无法看到您的明文密码
3. 支付信息全部交由微信支付/支付宝等第三方处理，平台不存储支付密码和银行卡信息
4. 用户数据按账号隔离，A 用户无法访问 B 用户的数据
5. 遵循最小必要原则，只收集服务所必需的个人信息
具体隐私政策请查阅「个人中心」→「隐私政策」。

**用户之间的数据安全隔离吗？**
是的。平台采用严格的账号级数据隔离机制：每位用户只能查看自己的订单、购物车、收货地址、售后记录等数据，无法访问其他用户的任何信息。系统后端会校验请求中的用户身份（通过 JWT Token），确保数据访问权限精确到个人。

**平台的商品信息多久更新一次？**
商品信息（价格、库存、描述等）由商家维护并实时同步。您在商品详情页看到的价格和库存即为当下实时数据。但考虑到网络延迟和系统缓存，极端情况下可能存在秒级延迟。加入购物车或下单时会再次校验实际的库存和价格。

**什么是优惠券？如何使用？**
优惠券是平台和商家发放的促销工具：
1. 优惠券类型：满减券（满 X 减 Y）、折扣券（打 X 折）、免邮券
2. 获取方式：活动页面领取、新人礼包、商家发放、平台赠送
3. 使用条件：每张优惠券有具体的使用门槛和适用范围
4. 使用方式：在结算页面选择可用优惠券即可，系统自动计算最优组合
5. 有效期：每张优惠券有明确的有效期，过期失效
优惠券不可转让、不可兑换现金、不可找零。具体规则见各优惠券的说明。

**不同优惠可以叠加使用吗？**
优惠叠加规则如下：
1. 平台优惠券 + 商家优惠券：部分可叠加，以结算页面实际显示为准
2. 满减活动 + 优惠券：通常可叠加
3. 同一类型多张优惠券：同一订单只能使用一张同类型优惠券
4. 秒杀商品：通常不能使用优惠券（以活动说明为准）
结算页面会自动展示最优优惠组合，您无需手动计算。如有疑问，可在下单前咨询客服。

**平台对商家有什么要求？**
平台对所有入驻商家有以下基本要求：
1. 提供真实有效的营业执照和相关资质
2. 商品信息真实准确，不得虚假宣传
3. 按规定时效发货，不得无故延迟
4. 及时处理售后问题，遵守平台售后规则
5. 不得销售假冒伪劣商品
如果发现商家存在违规行为，您可以通过商品详情页的「举报」按钮或联系平台客服进行投诉。

**遇到商品质量问题怎么办？**
如果您收到的商品存在质量问题：
1. 拍照或录视频保留证据
2. 在「我的订单」中申请售后（退货退款/换货/维修）
3. 上传证据说明问题所在
4. 商家审核后按售后流程处理
5. 如果商家不配合，可申请平台介入
质量问题属于商家的责任，退货邮费由商家承担。严重质量问题可联系平台客服要求对商家进行处理。

**如何联系人工客服？**
您可以通过以下方式联系人工客服：
1. 在任意页面点击「客服」图标，输入您的问题
2. AI 智能客服会先尝试解答，如果无法解决会提供「转人工」选项
3. 人工客服在线时间为每天 9:00-22:00
4. 非在线时间的留言会在次日按顺序回复
5. 紧急问题也可以在「我的订单」中直接对具体订单发起咨询

**如何举报违规商品或商家？**
您可以通过以下途径举报：
1. 商品详情页点击「举报」按钮，选择举报原因并提交
2. 联系客服提交举报信息
3. 举报类型包括：虚假宣传、假冒伪劣、价格欺诈、违禁商品等
平台会对举报进行调查核实，处理结果通过站内消息反馈给您。举报人的信息安全得到保护，不会被透露给被举报方。

**平台的交易规则有变更吗？**
平台规则会不定期更新优化。重大规则变更时：
1. 平台会通过站内消息推送通知
2. 在网站公告栏发布详细的变更说明
3. 涉及用户权益的变更会提前公告，给用户充分的了解和选择时间
建议定期关注平台公告。继续使用平台服务即表示同意最新的规则条款。

**商品可以囤货购入再转卖吗？**
平台面向终端消费者，不支持批量囤货和转卖行为。如果系统检测到异常的大批量采购行为（如短时间内大量购买同类商品），可能会：
1. 限制购买数量
2. 取消异常订单
3. 冻结账号
如果您有批发需求，请联系商家获取批发渠道信息，不要通过零售通道大量采购。

**订单纠纷会怎么处理？**
当买卖双方就订单产生纠纷时，处理流程如下：
1. 双方在线沟通协商（在售后详情页的聊天框进行）
2. 协商不成可申请平台介入
3. 平台客服核实双方证据和陈述
4. 根据平台规则和交易事实做出仲裁
5. 仲裁结果为最终结果，双方需按要求执行
建议保留所有沟通记录和相关凭证，这些在纠纷处理中非常重要。

**平台的服务费是多少？**
平台本身不向买家收取额外服务费。您支付的价格 = 商品价格 - 优惠金额 + 运费（如有）。部分特殊服务（如加急配送、礼品包装等）可能产生额外费用，但这些费用会在下单时明确标注。如果发现平台有未告知的收费，请联系客服核实。
